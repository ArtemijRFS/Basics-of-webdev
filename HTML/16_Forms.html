Forms - elements that collect user inputs

<form> - creates an HTML form for user input

<label> - defines a label for many form elements

<input> - input element of form. There are 5 main input types

Checkbox - for selecting zero or more of many choices

Select Yeat song

Text - single-line for text input

Type your favourite Yeat song

Radio - for selecting one of many

Your favourite singer

Select - a drop-down list

Textarea - multi-line input field

Button - for a clickable button

Submit - for submiting a form
