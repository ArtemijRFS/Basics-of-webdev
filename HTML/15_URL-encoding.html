URL - Universal Resource Locator - same for web address

Example - https://shop.monstercat.com/collections/all-products

Rule - scheme://prefix.domain:port/path/filename

  • scheme - type of Internet service(http oe https)

  • prefix - domain prefix(default for http is www)

  • domain - Internet domain name(like shop.monstercat.com)

  • port - port number at host(default for http is 80)

  • path - defines a path at server(he root directory of the site)

  • filename - the name of a document or resource
